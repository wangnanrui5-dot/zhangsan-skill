# 八维度风险评估标准

> 对任何内容进行性别议题风险扫描的评估框架。
> 八个维度独立评分0-10，≥7为高危。

---

## 维度一：女权/爱女敏感度

**定义：** 内容是否可能被解读为物化女性、不爱女、厌女、擦边。

**常见触发场景：**

| 行业 | 触发场景 |
|------|----------|
| 游戏 | 女角色暴露/挖洞设计、女角色沦为工具人、资源偏向男角色 |
| 美妆 | 用全男代言人、广告暗示女性需要"被改造"、产品名称物化女性 |
| 汽车 | 车展模特暴露、广告中女性只是"副驾"、"女司机"刻板印象 |
| 餐饮 | 广告文案性暗示、联名IP女角色被矮化 |
| 影视 | 女角色只是恋爱工具、"圣母""傻白甜"刻板塑造 |
| 科技 | 产品宣传暗示"女性不懂技术"、客服侮辱女性用户 |
| 服饰 | 过度性感营销、尺码歧视、广告中女性被物化 |
| 个人IP | 公开言论中的厌女倾向、翻车旧言论被挖出 |

**风险信号词：** 物化、不爱女、厌女、擦边、媚男、男凝、挖洞、露肉、工具人、花瓶、附属品

**评分标准：**

| 分数 | 含义 |
|------|------|
| 0-2 | 安全。内容性别中立或对女性有正向展现 |
| 3-4 | 低风险。有轻微可被解读的元素，但大概率不会被攻击 |
| 5-6 | 中风险。存在明确的可攻击点，但有辩护空间 |
| 7-8 | 高风险。大概率会被激进女权群体攻击，需要准备预案 |
| 9-10 | 极高风险。几乎必定引发大规模争议，建议修改后再发布 |

## 维度二：男权敏感度

**定义：** 内容是否可能被解读为媚女、忽视男性用户、对男性角色/产品线歧视。

**常见触发场景：**

| 行业 | 触发场景 |
|------|----------|
| 游戏 | 男角色建模/外观品质远低于女角色、资源全给女角色、男号"躺平" |
| 美妆 | 男性护肤品被嘲讽、"男人不需要化妆"刻板印象 |
| 汽车 | 过度强调"女性专属"设计、把男性排斥在目标用户外 |
| 影视 | 男角色全是反派/工具人、"男人都是渣"叙事 |
| 科技 | 产品宣传只针对女性、忽视男性用户需求 |
| 体育 | 男性运动员被忽视、女性赛事获得不对等的宣传资源 |

**风险信号词：** 媚女、田园女权、男号没人权、政治正确、女拳、区别对待、双标

**评分标准：**

| 分数 | 含义 |
|------|------|
| 0-2 | 安全。性别平衡或对男性有合理展现 |
| 3-4 | 低风险。男性用户可能有轻微不满，但不会形成舆情 |
| 5-6 | 中风险。男性用户群体可能集中吐槽 |
| 7-8 | 高风险。贴吧/虎扑/B站可能出现集中声讨 |
| 9-10 | 极高风险。可能引发男性用户大规模抵制 |

## 维度三：BL/腐向敏感度

**定义：** 内容是否涉及男性角色之间的暧昧互动，或可能触发BL粉丝的领地意识。

**常见触发场景：**

| 行业 | 触发场景 |
|------|----------|

| 出版 | 平台对BL内容限流/封禁（晋江事件）、BL vs BG资源分配 |
| 偶像 | 男团成员之间的CP营业、"拆CP"争议 |
| 动漫 | 官方CP vs 同人CP冲突、角色性取向争议 |

**风险信号词：** 卖腐、ooc、拆CP、腐蟑螂、套子、死小、掰弯、洗直、纯爱

**评分标准：**

| 分数 | 含义 |
|------|------|
| 0-2 | 安全。不涉及BL元素或处理得当 |
| 3-4 | 低风险。有轻微可被BL解读的元素 |
| 5-6 | 中风险。BL粉丝和反BL群体都可能有反应 |
| 7-8 | 高风险。可能引发BL vs BG大战 |
| 9-10 | 极高风险。多方混战，平台可能被迫表态 |

## 维度四：BG/异性恋敏感度

**定义：** 内容是否可能让异性恋CP党感到自己的CP被拆、被边缘化、被BL/GL覆盖。关注的是"他和她"这对CP关系是否被尊重。

**常见触发场景：**

| 行业 | 触发场景 |
|------|----------|
| 游戏 | 官方异性恋CP被拆散、BL同人声量盖过官方BG线、官方暧昧多线但不给BG确认 |
| 影视 | BG向原著被改编加入BL元素、男女主CP感被削弱、女主沦为配角 |
| 出版 | BG频道被BL内容入侵、平台推荐算法偏向BL |
| 偶像 | 男偶像"卖腐"让CP粉/女友粉不满 |
| 动漫 | 官方BG CP vs 同人BL CP冲突、角色关系被二创覆盖 |

**风险信号词：** 拆CP、官配、洗弯、be美学、异性恋没人权、强行掰弯

**评分标准：**

| 分数 | 含义 |
|------|------|
| 0-2 | 安全。异性恋CP关系清晰稳定 |
| 3-4 | 低风险。有轻微被边缘化的感觉 |
| 5-6 | 中风险。BG党可能集中表达不满 |
| 7-8 | 高风险。可能引发BG vs BL大战 |
| 9-10 | 极高风险。BG党大规模脱粉/抵制 |

## 维度五：ML/代入向敏感度

**定义：** ML（Master Love）指游戏中角色喜欢玩家主角的设定。核心体验是"角色爱我"。ML角色可男可女，关键是角色→主角的情感投射是否被破坏。

**常见触发场景：**

| 行业 | 触发场景 |
|------|----------|
| 游戏 | ML角色与其他角色产生暧昧（"共享"）、ML角色的情感线被削弱、主角存在感被降低 |
| 游戏 | ML角色之间互相组CP（角色不爱主角爱彼此）、官方在ML角色间搞暧昧 |
| 游戏 | 同一角色在不同线路/活动中对不同主角示好（"海王"感） |
| 动漫/IP | 原作ML设定在改编中被改为角色间CP |

**风险信号词：** 共享、海王、不专一、角色互CP、代入感破坏、我老公/老婆跟别人跑了

**评分标准：**

| 分数 | 含义 |
|------|------|
| 0-2 | 安全。ML体验完整，角色→主角情感线清晰 |
| 3-4 | 低风险。有轻微"被共享"的感觉 |
| 5-6 | 中风险。ML党可能不满，讨论区出现抱怨 |
| 7-8 | 高风险。ML党大面积愤怒，"塌房"情绪蔓延 |
| 9-10 | 极高风险。ML用户大规模弃游/弃坑 |

## 维度六：乙女敏感度

**定义：** 乙女指女性向恋爱游戏品类（女主×多男角色）。是BG的一个子集，但有独特的代入体验和雷区。乙女玩家代入女主，追求"男角色只爱我"的沉浸感。

**常见触发场景：**

| 行业 | 触发场景 |
|------|----------|
| 游戏 | 男角色与其他女角色有暧昧、女主被弱化/工具化、女主外观/人设不讨喜、男角色之间被搞BL |
| 游戏 | 乙女游戏加入男性向/BL向内容、多男角色资源分配不均（偏心某角色）|
| 游戏 | 联动/周边只出部分角色、节日活动缺角色 |
| IP衍生 | 乙女IP改编时女主被替换/削弱、男角色人设崩坏 |

**风险信号词：** 女主透明、工具人女主、偏心、不公平、塌房、共享男友、净土、乙女法

**评分标准：**

| 分数 | 含义 |
|------|------|
| 0-2 | 安全。乙女体验完整，女主有存在感，男角色专一 |
| 3-4 | 低风险。有轻微沉浸感破坏 |
| 5-6 | 中风险。乙女玩家开始在社区表达不满 |
| 7-8 | 高风险。可能引发"净土保卫战"、大规模差评 |
| 9-10 | 极高风险。乙女玩家大规模退坑/退款 |

## 综合评估规则

1. **总体风险等级** = 所有维度中最高分决定
   - 任一维度 ≥ 7 → 🔴 高风险
   - 所有维度 < 7 但有 ≥ 5 → 🟡 中风险
   - 所有维度 < 5 → 🟢 低风险

2. **多维度叠加** = 风险倍增
   - 如果2个以上维度 ≥ 5，即使没有单项≥7，也应升级为🔴高风险
   - 因为多维度同时触发意味着多个群体同时被激怒，局面更难控制

3. **行业系数**
   - 女性用户占比高的行业（美妆/乙女游戏/服饰），女权维度权重×1.5
   - 男性用户占比高的行业（FPS/体育/汽车改装），男权维度权重×1.5
   - 同人文化活跃的行业（游戏/动漫/影视），BL/BG/GL维度权重×1.5
   - 有ML/乙女机制的游戏，ML和乙女维度权重×1.5
