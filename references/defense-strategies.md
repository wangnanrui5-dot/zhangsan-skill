# 防守策略池

> 不是固定菜单，是弹药库。根据具体事件自由组合。

---

## 策略1：跳出框架
- **定义：** 制造"二元对立"之外的第三选项，让争论失焦
- **适用：** 被扣帽子（媚男/媚女/不爱女等）
- **示例框架：** "不媚男不媚女只媚钱" / "我们不站任何一边，我们站用户体验"
- **风险：** 两边都可能觉得你在和稀泥

## 策略2：双标反击
- **定义：** 用攻击方自己的逻辑打回去
- **适用：** 攻击方存在明显双标时
- **示例框架：** "只许X不许Y？" / "按这个标准，你们自己的XX也不合格"
- **风险：** 可能被认为在转移话题

## 策略3：制造分裂
- **定义：** 让被代表的群体内部反驳，拆散攻击方阵营
- **适用：** 攻击方自称代表某群体但实际不代表大多数
- **示例框架：** "别代表全体XX，我就是XX我就喜欢" / 让温和派跟极端派吵
- **风险：** 需要真的有内部分歧才能用

## 策略4：玩梗稀释
- **定义：** 在舆情起来之前把话题定性为搞笑梗，消解严肃性
- **适用：** 预防性使用，舆情还没爆时
- **示例框架：** 提前铺搞笑帖，让围观者第一反应是笑而不是愤怒
- **风险：** 如果是严肃实锤问题，玩梗会被骂"不尊重"

## 策略5：甩给第三方
- **定义：** 把火力引向无法反击的目标
- **适用：** 联动/IP/代言类争议
- **示例框架：** "角色设定是原著定的" / "这是合作方的要求"
- **风险：** 可能影响合作关系

## 策略6：数据打脸
- **定义：** 用投票/销量/付费/调研数据直接证伪攻击方论点
- **适用：** 攻击方基于错误认知或伪需求
- **示例框架：** "投票数据显示XX垫底" / "付费数据显示真正买单的是XX群体"
- **风险：** 数据可能被质疑来源或cherry-pick

## 策略7：升维到现实
- **定义：** 把线上争议类比到现实荒谬场景，让围观者瞬间站队
- **适用：** 标准被无限扩大化时
- **示例框架：** "按这个标准，夏天穿吊带出门就是XX" / "举报动画片带坏小孩"
- **风险：** 类比不当会被反驳"false equivalence"

## 策略8：捧杀
- **定义：** 表面夸对方，实际把标准拉高到对方自己打脸
- **适用：** 攻击竞品的好评帖、或抬高攻击方的标准让其自相矛盾
- **示例框架：** "好棒！那XX方面是不是也这么重视呢？" / 表面夸紧跟一刀
- **风险：** 太明显会被识破

## 策略9：冷处理+替代热点
- **定义：** 不回应，用新话题转移注意力
- **适用：** 热度不高 / 回应会火上浇油 / 大部分性别议题72小时内自然降温
- **示例框架：** 同时发布新产品/新活动/新内容抢占注意力
- **风险：** 如果是实锤问题，冷处理会被骂"装死"

## 策略10：借刀杀人
- **定义：** 引导第三方（KOL/媒体/意见领袖）替你发声
- **适用：** 官方不方便直接下场时
- **示例框架：** 找友好KOL发理性分析帖 / 提供素材给媒体做客观报道
- **风险：** KOL可能翻车或被识别为"水军"

## 策略11：共情瓦解
- **定义：** 站在攻击方阵营内部说话，把温和派从极端派里摘出来
- **适用：** 攻击方不是铁板一块，内部有分歧
- **示例框架：** "我也是XX群体的，但我觉得这次确实过了" / 以内部人身份发声
- **风险：** 身份伪装被识破

## 策略12：历史翻篇
- **定义：** 找到攻击方过去自相矛盾的言论打回去
- **适用：** 攻击方有可查的历史双标言论
- **示例框架：** 截图对比"你去年说XX，今年说YY"
- **风险：** 可能被说"翻旧账没意义"

## 策略13：受害者叙事
- **定义：** 把被攻击方转化为受害者身份，抢占道德制高点
- **适用：** 攻击方用语过激、存在霸凌行为
- **示例框架：** "我只是说了句喜欢就被骂XX" / "说好话都要被扣帽子"
- **风险：** 过度卖惨会被嘲讽

## 策略14：阵营嫁接
- **定义：** 把攻击者绑定到更不受欢迎的群体上
- **适用：** 攻击方的言论跟某个臭名昭著的群体/事件相似
- **示例框架：** "你这说法跟XX一模一样" / 让围观者自动把攻击方归类为极端群体
- **风险：** 嫁接不当会被反驳

## 策略15：沉默大多数动员
- **定义：** 写帖子让不敢说话的人站出来
- **适用：** 社区存在"只能骂不能夸"的氛围，沉默大多数被压制
- **示例框架：** "是不是只有我觉得其实还不错？" / "说喜欢就被扣帽子，我不服"
- **风险：** 如果沉默大多数实际上也不满，会适得其反

## 策略选用参考

| 场景 | 推荐策略组合 |
|------|-------------|
| 被扣"不爱女"帽子 | 数据打脸 + 沉默大多数动员 + 受害者叙事 |
| 被骂"媚男/媚女" | 跳出框架 + 双标反击 |
| BL vs BG内战 | 制造分裂 + 冷处理 |
| ML角色被"共享" | 共情瓦解 + 受害者叙事 |
| 乙女"净土保卫战" | 沉默大多数动员 + 制造分裂 |
| 竞品好评帖需要打 | 捧杀 + 升维到现实 |
| 联动IP引发争议 | 甩给第三方 + 玩梗稀释 |
| 实锤翻车 | 受害者叙事（如果被攻击过度）/ 冷处理 + 替代热点 |
| 舆情刚起来 | 玩梗稀释 + 借刀杀人（预防）|
| 舆情已经爆了 | 冷处理 + 替代热点 + 共情瓦解 |

**以上只是参考，实战中根据具体情况自由组合，所有适用的策略都应列出，由使用者自行裁剪。**
