# 测试用例 3：餐饮竞品攻击

## 输入

**行业：** 茶饮
**路径：** C（竞品攻击话术）

**事件速录：**
```
竞品名称/产品：某头部茶饮品牌
事件/内容：竞品联名某热门动漫IP，首批周边只有男角色杯套，女角色一个都没有
竞品用户男女比：3:7（女性为主）
竞品已知的性别争议历史：之前联名也被说过"重男轻女"
竞品社区内部有哪些势力/裂痕：女用户群体 vs 男角色粉 vs IP原著粉
期望效果：降口碑 + 制造内部分裂
```

## 期望输出要点

- 主攻维度应为"女权/爱女"
- 应利用历史旧账（"又来了"）
- 应输出多个伪装身份方向（至少：女用户、路人、竞品老用户）
- 攻击方向应≥4个
- 应提醒：不要在攻击中提自家品牌
- 应有反识破要点
