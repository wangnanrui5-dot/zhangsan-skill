# 测试用例 2：美妆品牌代言人对冲

## 输入

**行业：** 美妆（女性护肤品）
**用户男女比：** 1:9（几乎全女性）
**路径：** B（对冲话术）

**事件速录：**
```
事件/内容：品牌官宣男性偶像团体为新品代言人
攻击方是谁：小红书女性用户、爱女博主
主要平台：小红书、微博
当前热度：正爆
有无实锤：半实锤（确实是全男代言）
品牌方已有回应：无
可辩护的点：该偶像粉丝购买力强，往期联名销量好
```

## 期望输出要点

- 应识别为"不爱女"类攻击
- 策略应包含但不限于：数据打脸（粉丝购买力数据）、沉默大多数动员、共情瓦解
- 不应建议"冷处理"（热度正爆+有实锤基础）
- 话术方向应≥5个
- 应提醒风险：数据打脸可能被反驳"买单的是男粉不是女用户"
