# 测试用例 1：游戏男角色PV风险分析

## 输入

**行业：** 古风MMO游戏
**用户男女比：** 4:6（女性略多）
**路径：** A（风险分析）

**内容描述：**
我们准备发一个男角色的角色PV，3分钟，画面精美，有大量男角色近景特写和战斗场景。同期没有女角色内容计划。

## 期望输出要点

- 总体风险：🟡中 或 🔴高
- 女权/爱女维度应≥5（资源给男角色、女性用户可能觉得"不爱女"）
- 男权维度应≤3（男角色得到资源，男性用户不会不满）
- BL维度应≥4（男角色近景特写可能被BL解读）
- 应建议：同期或近期安排女角色内容平衡
- 应预测：小红书可能出现"又给男号做PV""女号什么时候有"类帖子
